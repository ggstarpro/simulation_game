# 運用例
* 【初期状態】
指数（ATH）: QQQ 400pt / SOX 5000pt
ETF価格: QLD $100 / SOXL $50
あなたの行動: 規約通り、DD 0% のルール（QLD 90% : SOXL 10%）で買い付けます。
```
項目,QLD（90%）,SOXL（10%）,合計
保有金額,900万円,100万円,"1,000万円"
```

* 【フェーズA】調整下落の始まり（DD -20%）
指数の状態: QQQ 340pt (-15%) / SOX 4000pt (-20%)
最大ドローダウン（DD）: SOXの -20% が適用。
ETF価格（推定）: QLD $68 (-32%) / SOXL $20 (-60%)

```
資産評価額（何もしない状態）:
   QLD: 900万 × (68/100) = 612万円
   SOXL: 100万 × (20/50) = 40万円
   総資産 = 652万円 （現状比率 QLD 94% : SOXL 6%）

規約の判定: DD -20%のルールに従い、目標比率を「QLD 60% : SOXL 40%」にする。

【あなたのアクション：リバランス実行】
   総資産652万円に対する目標額は、QLD 391万円、SOXL 261万円です。
   よって、QLDを221万円分売却し、SOXLを221万円分買い増します。
```


* 【フェーズA】絶望の大暴落・セリングクライマックス（DD -35%）
指数の状態: QQQ 300pt (-25%) / SOX 3250pt (-35%)
最大ドローダウン（DD）: SOXの -35% が適用。
ETF価格（推定）: QLD $50 (-26%) / SOXL $8 (-60%)

```
資産評価額（何もしない状態）:
   QLD: 391万 × (50/68) = 287万円
   SOXL: 261万 × (8/20) = 104万円
   総資産 = 391万円 （現状比率 QLD 73% : SOXL 27%）

規約の判定: DD -35%のルールに従い、目標比率を「QLD 10% : SOXL 90%」にする。

【あなたのアクション：極限のリバランス実行】
   総資産391万円（初期から-60%）という地獄のような口座残高を見ながら、目標額（QLD 39万円、SOXL 352万円）に合わせます。
   恐怖に耐え、QLDを248万円分売却し、血まみれのSOXLを248万円分買い増します。これでSOXLの保有額は352万円（比率90%）となります。
```

* 【フェーズB】急回復・V字反発（DD -5%）

最大ドローダウン（DD）: QQQの -5%。（まだATHは超えていない）
ETF価格（推定）: QLD $75 (+50%) / SOXL $18 (+125%)

```
資産評価額:
   QLD: 39万 × (75/50) = 58万円
   SOXL: 352万 × (18/8) = 792万円
   総資産 = 850万円 （現状比率 QLD 7% : SOXL 93%）

規約の判定: まだ過去最高値を奪還していないため、「フェーズB：回復局面」に該当。

【あなたのアクション：完全放置】
   底値で仕込んだSOXLが火を噴き、資産が急回復しました。現状のSOXL比率は93%にも達しており「少し利確したい」という誘惑に駆られますが、規約通り一切売買せずホールド（放置）します。
```

* 【フェーズC】最高値更新・AIバブル到来（UP +50%）

```

   総資産は初期の2.4倍に膨れ上がりました。現在のSOXL比率は97%であり、上限の90%を超過しています。
   総資産2,445万の90%である「2,200万円」をSOXLの目標額とし、超過している約176万円分のSOXLを売却して利確。その資金でQLDを買い、リスクを退避させます。

```
