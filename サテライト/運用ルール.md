# 【最高値ドローダウン制御型】QLD・SOXL リバランス運用規約

本規約は、最終資産額（CAGR）の最大化を絶対目標とし、
QLD（Nasdaq100 2倍）とSOXL（半導体株 3倍）を用いたダイナミック・アセットアロケーション戦略の実行手順を定めたものである。
※ 短期的なボラティリティや大きなドローダウンは戦略上の「コスト」として許容する。

## 1. 運用の基本原則（鉄の掟）
1. **投資対象:** QLDおよびSOXLのみ。配当はすべて再投資する。
2. **判定頻度:** **最低毎月1回、最終営業日の引け間際のみ。理想は週1回。**日々の値動きやノイズ（ニュース、アナリストの予測）は一切無視する。
3. **判断基準:** 「QQQ or SOXの週間終値ベースの過去最高値（ATH）」からの下落率（DD）、上昇率（UP）、および長期テクニカル指標のみをシグナルとする。

---

## 2. フェーズ別運用ルール
DDを確認し、以下の3フェーズのいずれかのアクションを実行する。

### 【フェーズA：下落局面】
**状態:** QQQまたはSOXが過去最高値（ATH）を更新できず、かつ特例を満たしていない状態。
**行動:** 下落率に基づき、以下の比率へリバランスを行う。(※「QLD ➔ SOXL 」のみ、フェーズAのテーブルは「SOXLの最低目標比率」です)

| DD | QLD比率 | SOXL比率 |
| :--- | :---: | :---: |
| **0% ～ -9.9%** | 90% | 10% |
| **-10% ～ -14.9%** | 80% | 20% |
| **-15% ～ -19.9%** | 70% | 30% |
| **-20% ～ -24.9%** | 60% | 40% |
| **-25% ～ -29.9%** | 40% | 60% |
| **-30% ～ -34.9%** | 20% | 80% |
| **-35% ～ -39.9%** | 10% | 90% |
| **-40% 以上** | 0% | 100% |

※ DD = max(QQQの週末終値ATHからの下落率, SOXの週末終値ATHからの下落率)
※ 最も下落している（DDの数値が大きい）指数の数値を適用する。

### 特例
**状態:** 市場が歴史的なセリング・クライマックス（大底）に達したと判定される状態。
**行動:** 以下の「クライマックス条件」の いずれか1つ を満たした場合、即座に 【 QLD 0% : SOXL 100% 】 へリバランスする。
**優先度:** フェーズAのドローダウン率（DD）テーブルより優先して適用される。

- 条件1（RSI極値）: QQQ または SOX の「月足RSI（期間14）」が 40.0 以下 を記録した。
- 条件2（対数長期サポート）: QQQ または SOX の週足チャート（対数スケール）において、2008年以降の大底を結んだ長期サポートラインにタッチ、または下回った（ザラ場のヒゲを含む）。

```
※長期サポートライン作成のための基準座標
QQQ: (2010/6/28: 41.78), (2016/2/8: 94.98), (2018/12/24: 144.86), (2020/3/16: 173.28), (2022/12/27: 260.82)
SOX: (2020/03/16: 1286.43), (2022/10/10: 2180.26)
※特例が発動した後は、フェーズB（回復局面）へ移行するまで「SOXL 100%」をホールドする。
```

### 【フェーズB：回復局面】
**状態:** 下落局面（フェーズA）から反転したが、まだ旧最高値を奪還していない状態。
**行動:** **現状の比率を維持（ホールド）する。** ATH更新までは一切売買は行わない。
**例外** 直近の判定時よりもDDが悪化し、かつ現在のSOXL保有比率がフェーズAの規定比率を下回っている場合のみ、フェーズAのテーブルへ回帰して買い増しを実行する。

### 【フェーズC：新高値更新後】
**状態:** QQQまたはSOXが週末終値ベースでATHを更新した状態。
**行動:** 「SOXL保有上限（Cap）」テーブルを適用し、リスクの段階的退避を行う。(※SOXL ➔ QLD のみ、現在の比率がテーブルより高い場合のみ利確します。)


**【キャップ適用ルール】**
* **利確（退避）:** 「現在のSOXL比率」が「上限値（Cap）」を 超えている場合のみ、その上限値までSOXLを売却し、QLDを買い増す。
* **放置（順張り）:** 「現在のSOXL比率」が「上限値（Cap）」以下の場合は、何もしない（完全放置）。目標比率へ引き上げるための買い増しは禁止。

| UP | SOXLの保有上限（Cap） |
| :--- | :---: |
| **0% ～ +49.9%** | ※フェーズBの比率を現状維持・利確禁止 |
| **+50% ～ +74.9%** | 90% |
| **+75% ～ +99.9%** | 80% |
| **+100% ～ +124.9%** | 70% |
| **+125% ～ +149.9%** | 60% |
| **+150% ～ +174.9%** | 50% |
| **+175% ～ +199.9%** | 40% |
| **+200% ～ +224.9%** | 30% |
| **+225% ～ +250.9%** | 20% |
| **+250% 以上** | 10% |

※ UP = max(QQQのATH更新後上昇率, SOXのATH更新後上昇率)
※ 最も上昇している（UPの数値が大きい）指数の数値を適用する。
※ 「ATH更新後も、まず+50%上昇に達するまでは絶対にSOXLを削らない」点に留意。


#### 例：フェーズCで「UP +120%」までバブルが進行した
ルールに従い、あなたのポートフォリオは「QLD 30% : SOXL 70%」になっています。
直後、バブルが弾けて高値から「DD -30%」の暴落（フェーズA）が来た
SOX指数が-30%下落すると、SOXL（3倍）は株価が約 -70% ほど吹き飛びます。
価格変動により、あなたのポートフォリオの実際の比率は「QLD 55% : SOXL 45%」などに大きく偏ります。
ここでフェーズA（-30%）のテーブルを見ます。ルール上の最低目標比率は「SOXL 80%」です。
あなたは、温存しておいたQLDを大量に売り、暴落して安くなったSOXLを一気に買い増して「SOXL 80%」に引き上げます。

#### 例：フェーズCの初期（UP +20%）で騙しに遭い、暴落した
あなたのポートフォリオは「SOXL 100%」のままです。
そのまま「DD -30%」の暴落が来ました。
フェーズA（-30%）の最低目標比率は「SOXL 80%」です。しかし、あなたは現在100%（価格変動を加味しても80%以上）を持っています。
一方通行ルールにより、SOXLを売って80%に下げる（損切りする）ことはしません。「完全ホールド」で耐え抜きます。


## 3. 運用ロジックの要点（クオンツ・アナリストによる補足）
1. **歴史的クライマックス判定（特例）の意義:**
  - 暴落時、下落率（DD）が-40%に達する前に相場が反発してしまう「機会損失」を防ぎます。
  - 月足RSI40割れ、あるいは対数長期トレンドラインへの接触は、歴史が証明する「究極の恐怖（大底）」のシグナルです。ここで強制的にSOXL 100%を叩き込むことで、その後の回復フェーズにおけるCAGRが飛躍的に向上します。
2. **なぜ「目標」ではなく「上限（Cap）」なのか:**
   - 不必要な買い増しによる「高値掴み」を完璧に防ぐためです。
   - 大底で仕込んだSOXLの爆発力を、上昇局面でも「上限に達するまで」決して手放さず、相場が過熱（+50%以上）した時にのみ段階的に利確（QLDへ退避）を行います。
2. **なぜ「放置」が戦略に含まれるのか:**
   - ボラティリティの激しいレバレッジETF運用において、最もリターンを低下させるのは「微益撤退」と「狼狽売り」です。
   - フェーズB（回復）とフェーズCの上限割れ時に「何もしない」ことを明文化することで、現在持っている強気ポジションを数学的に最も有利な形で維持できます。
3. **実運用の極意:**
   - 判定日に迷ったときは、以下の2つだけを自問してください。
     - 「今は歴史的な大底（特例）か？」 ➔ YesならSOXL100%へ。
     - 「（フェーズCにおいて）今のSOXL比率は、設定された上限を超えているか？」 ➔ Noなら無視して放置。Yesなら上限まで機械的に利確。
   - これだけで、人間の脆弱な「感情の介入」を完全に排除したシステマティック・トレードが完成します。
