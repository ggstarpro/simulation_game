# 【最高値ドローダウン制御型】QLD・SOXL リバランス運用規約

本規約は、最終資産額（CAGR）の最大化を絶対目標とし、QLD（Nasdaq100 2倍）とSOXL（半導体株 3倍）を用いたダイナミック・アセットアロケーション戦略の実行手順を定めたものである。

- ※重要事項
  - 短期的なボラティリティや大きなドローダウンは戦略上の「コスト」として許容する。
  - ルールの判定基準である「ATH（過去最高値）」「下落率（DD）」「上昇率（UP）」は、必ず**「QQQ（NASDAQ100指数）」または「SOX指数」**そのもののチャートを見て判定すること。
  - 絶対に「QLD」や「SOXL」のETF価格を基準にしてはいけない。（ETFの価格は減価によって長期的に歪むため、ATHの基準値として使うとシステムが誤作動を起こす）。

## 1. 運用の基本原則
1. **投資対象:** QLDおよびSOXLのみ。配当はすべて再投資する。
2. **判定頻度:** 最低毎月1回、最終営業日の引け間際のみ（理想は週1回）。日々の値動きやノイズ（ニュース、アナリストの予測）は一切無視する。
3. **判断基準:** 「QQQ または SOX の週間終値ベースの過去最高値（ATH）」からの下落率（DD）、上昇率（UP）、および長期テクニカル指標のみをシグナルとする。

## 2. フェーズ別運用ルール
各判定日にDD（ドローダウン）またはUP（上昇率）を確認し、以下の3フェーズのいずれかのアクションを実行する。

### 【フェーズA：下落局面】
**状態:** QQQまたはSOXが過去最高値（ATH）を更新できず、かつ特例を満たしていない状態。
**行動:** 下落率に基づき、以下の比率へリバランスを行う。（※アクションは「QLD ➔ SOXL の買い増し」のみ。フェーズAのテーブルは「SOXLの最低目標比率」を示す）

| DD | QLD比率 | SOXL比率 | 意図 |
| :--- | :---: | :---: | :---: |
| **0% ～ -9.9%** | 90% | 10% | 【ノイズ・静観】 日常的な変動。基本的には動かない。 |
| **-10% ～ -14.9%** | 80% | 20% | 【買い下がり開始（緩）】 軽い調整。ここから10%ずつ慎重に拾い始める。 |
| **-15% ～ -19.9%** | 70% | 30% | 同上。通常の調整ライン。 |
| **-20% ～ -24.9%** | 60% | 40% | 同上。警戒感が高まる水準。着実に弾（SOXL）を込めていく。 |
| **-25% ～ -29.9%** | 40% | 60% | 【攻撃加速ゾーン（急）】 恐怖が市場を支配し始める領域。ここからギアを上げ、20%単位で大きく買い向かう。 |
| **-30% ～ -34.9%** | 20% | 80% | 同上。多くの投資家がパニックで投げ売りする水準。手加減せず強気に拾う。 |
| **-35% ～ -39.9%** | 10% | 90% | 【最終仕上げゾーン】 大底に極めて近い水準。最後の余力を残しつつほぼ全力を投入。 |
| **-40% 以上** | 0% | 100% | 【最大攻撃ライン】 SOXの深い調整の極致。ここで全資金をSOXLに固定し、歴史的反発を丸取りする。 |

【適用ルール】
DD = max(QQQの週末終値ATHからの下落率, SOXの週末終値ATHからの下落率)。最も下落している（DDの数値が大きい）指数の数値を適用する。

### 特例（セリング・クライマックス）
**状態:** 市場が歴史的な大底に達したと判定される状態。
**行動:** 以下のいずれかを満たした場合、DDテーブルを無視して即座に 【 QLD 0% : SOXL 100% 】 へリバランスし、フェーズB（回復）へ移行するまでホールドする。
- 条件1（RSI極値）: QQQ または SOX の「月足RSI（期間14）」が 40.0 以下 を記録。
- 条件2（対数長期サポート）: QQQ または SOX の週足チャート（対数スケール）において、2008年以降の大底を結んだ長期サポートラインにタッチ、または下回った（ザラ場のヒゲを含む）。

```
※ 長期サポートライン作成のための基準座標
QQQ: (2010/6/28: 41.78), (2016/2/8: 94.98), (2018/12/24: 144.86), (2020/3/16: 173.28), (2022/12/27: 260.82)
SOX: (2020/3/16: 1286.43), (2022/10/10: 2180.26)
※ 特例が発動した後は、フェーズB（回復局面）へ移行するまで「SOXL 100%」をホールドする。
```

### 【フェーズB：回復局面】
**状態:** 下落局面（フェーズA）から反転したが、まだ旧最高値を奪還していない状態。
**行動:** **現状の比率を維持（ホールド）する。** ATH更新までは一切売買は行わない。
**例外** 直近の判定時よりもDDが悪化（二番底など）し、かつ現在のSOXL保有比率がフェーズAの規定比率を下回っている場合のみ、フェーズAのテーブルへ回帰して買い増しを実行する。

### 【フェーズC：新高値更新後】
**状態:** QQQまたはSOXが週末終値ベースでATHを更新した状態。
**行動:** 以下の「SOXL保有上限（Cap）」テーブルを適用し、リスクの段階的退避を行う。（※アクションは「SOXL ➔ QLD の利確」のみ。現在のSOXL比率がテーブルより高い場合のみ利確する。）

| UP | SOXLの保有上限（Cap） | 意図 |
| :--- | :---: | :---: |
| **0% ～ +49.9%** | 維持 | 【最大利益ゾーン】 フェーズBの比率を現状維持・利確禁止。直近高値からのブレイクアウト後、最も到達確率が高く一直線に伸びやすい領域。SOXLの複利を最大化する。 |
| **+50.0% ～ +64.9%** | 90% | 【リスクオフ開始】 到達確率が下がり始める+50%から、なめらかに退避を開始。 |
| **+65.0% ～ +79.9%** | 80% | 同上。15%上昇ごとに10%ずつ慎重に落とし、境界線での「往復ビンタ（無駄な売買摩擦）」を防ぐ。 |
| **+80.0% ～ +94.9%** | 70% | 同上。高い利益水準をキープしつつ、次の暴落用の「弾」を静かに装填していく。 |
| **+95.0% ～ +109.9%** | 50% | 【利確加速ゾーン（急）】 指数が約2倍（+100%）となる歴史的な節目。暴落リスクが跳ね上がるため、ここから一気に20%単位でQLDへ逃がし、防御を固める。 |
| **+110.0% ～ +124.9%%** | 30% | 同上。バブル崩壊の直撃を避けるための「緊急脱出」を完了させる。 |
| **+125.0% ～ +169.9%** | 20% | 【ノイズ吸収・バブル追従ゾーン】 防御はすでに完了しているため、帯（バンド）を広く取り高値圏の激しいノイズを無視する。歴史的バブルの「最後の一滴」まで20%の火力で付き合う。 |
| **+170.0% 以上** | 10% | ここから先のチキンレースには付き合わず、事実上の完全リスクオフ状態で下落を待つ。 |

**【キャップ適用ルール】**
UP = max(QQQのATH更新後上昇率, SOXのATH更新後上昇率)。最も上昇している指数の数値を適用する。「ATH更新後も、まず+50%上昇に達するまでは絶対にSOXLを削らない」点に留意。
* **利確（退避）:** 「現在のSOXL比率」が「上限値（Cap）」を超えている場合のみ、その上限値までSOXLを売却し、QLDを買い増す。
* **放置（順張り）:** 「現在のSOXL比率」が「上限値（Cap）」以下の場合は、何もしない（完全放置）。目標比率へ引き上げるための買い増しは禁止。

#### 運用シミュレーション例1：フェーズCで「UP +120%」までバブルが進行した直後に暴落
ルールに従い、あなたのポートフォリオは「QLD 70% : SOXL 30%」になっています（※+120%のSOXL上限Capが30%のため）。
直後、バブルが弾けて高値から「DD -30%」の暴落（フェーズA）が来ました。
SOX指数が-30%下落すると、SOXL（3倍）は株価が約-70%ほど吹き飛び、QLD（2倍）も約-60%下落します。
価格変動により、あなたのポートフォリオの実際の比率は概算で「QLD 76% : SOXL 24%」などに偏ります。
ここでフェーズA（-30%）のテーブルを見ます。ルール上の最低目標比率は「SOXL 80%」です。
あなたは、温存しておいたQLDを大量に売り、暴落して安くなったSOXLを一気に買い増して「SOXL 80%」に引き上げます。

#### 運用シミュレーション例2：フェーズCの初期（UP +20%）で騙しに遭い、暴落した
フェーズCの最大利益ゾーン（Cap維持）のため、あなたのポートフォリオは「SOXL 100%」のままです。
そのまま「DD -30%」の暴落が来ました。
フェーズA（-30%）の最低目標比率は「SOXL 80%」です。しかし、あなたは現在100%（価格変動を加味しても80%以上の高比率）を持っています。
一方通行ルールにより、SOXLを売って80%に下げる（損切りする）ことはしません。「完全ホールド」で耐え抜きます。


## 3. 運用ロジックの要点（クオンツ・アナリストによる補足）
1. **歴史的クライマックス判定（特例）の意義:**
  - 暴落時、下落率（DD）が-40%に達する前に相場が反発してしまう「機会損失」を防ぐ。
  - 月足RSI40割れ、あるいは対数長期トレンドラインへの接触は、歴史が証明する「究極の恐怖（大底）」のシグナルである。ここで強制的にSOXL 100%を叩き込むことで、その後の回復フェーズにおけるCAGRが飛躍的に向上する。
2. **なぜ「目標」ではなく「上限（Cap）」なのか:**
   - 不必要な買い増しによる「高値掴み」を完璧に防ぐためである。
   - 大底で仕込んだSOXLの爆発力を、上昇局面でも「上限に達するまで」決して手放さず、相場が過熱（+50%以上）した時にのみ段階的に利確（QLDへ退避）を行う。
2. **なぜ「放置」が戦略に含まれるのか:**
   - ボラティリティの激しいレバレッジETF運用において、最もリターンを低下させるのは「微益撤退」と「狼狽売り」である。
   - フェーズB（回復）とフェーズCの上限割れ時に「何もしない」ことを明文化することで、現在持っている強気ポジションを数学的に最も有利な形で維持できる。
3. **実運用の極意:**
   - 判定日に迷ったときは、以下の2つだけを自問してください。
     - 「今は歴史的な大底（特例）か？」 ➔ YesならSOXL100%へ。
     - 「（フェーズCにおいて）今のSOXL比率は、設定された上限を超えているか？」 ➔ Noなら無視して放置。Yesなら上限まで機械的に利確。
