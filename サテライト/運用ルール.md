# 【最高値ドローダウン制御型】QLD・SOXL リバランス運用規約

本規約は、最終資産額（CAGR）の最大化を絶対目標とし、
QLD（Nasdaq100 2倍）とSOXL（半導体株 3倍）を用いたダイナミック・アセットアロケーション戦略の実行手順を定めたものである。
※ 短期的なボラティリティや大きなドローダウンは許容する。

## 1. 運用の基本原則（鉄の掟）
1. **投資対象:** QLDおよびSOXLのみ。配当はすべて再投資する。
2. **判定頻度:** **最低毎月1回、最終営業日の引け間際のみ。理想は週1回**。日々の値動きやニュースは一切無視する。
3. **判断基準:** 「QQQ or SOXの 週間終値ベースの過去最高値（ATH）」からの下落率、および上昇率のみをシグナルとする。

---

## 2. フェーズ別運用ルール
DDを確認し、以下の3フェーズのいずれかのアクションを実行する。

### 【フェーズA：下落局面】
**状態:** QQQ or SOXが過去最高値を更新できず、下落している状態。
**行動:** 下落率に基づき、以下の比率へリバランスを行う。

| DD | QLD比率 | SOXL比率 |
| :--- | :---: | :---: |
| **0% ～ -9.9%** | 90% | 10% |
| **-10% ～ -14.9%** | 80% | 20% |
| **-15% ～ -19.9%** | 70% | 30% |
| **-20% ～ -24.9%** | 60% | 40% |
| **-25% ～ -29.9%** | 40% | 60% |
| **-30% ～ -34.9%** | 20% | 80% |
| **-35% ～ -39.9%** | 10% | 90% |
| **-40% 以上** | 0% | 100% |

※DD=max(QQQの週末終値ATHからの下落率,SOXの週末終値ATHからの下落率)

### 【フェーズB：回復局面】
**状態:** 下落局面（フェーズA）から反転したが、まだ旧最高値を奪還していない状態。
**行動:** **現状の比率を維持（ホールド）する。** ATH更新までは一切売買は行わない。
*※例外：前月の週間終値を割り込んだ場合、フェーズAのテーブルへ回帰する。*

### 【フェーズC：新高値更新後】
**状態:** QQQまたはSOXが週末終値ベースATHを更新した状態。
**行動:** 以下の「SOXL保有上限（Cap）」を適用する。

**【重要：キャップ適用ルール】**
* **ルール:** 「現在のSOXL比率」が「上限値」を超えている場合のみ、その上限値まで売却して調整する。
* **放置:** 「現在のSOXL比率」が「上限値」以下の場合は、**何もしない（放置）**。

| UP | SOXLの保有上限（Cap） |
| :--- | :---: |
| **0% ～ +49.9%** | （※フェーズBの比率を現状維持） |
| **+50% ～ +74.9%** | 90% |
| **+75% ～ +99.9%** | 80% |
| **+100% ～ +124.9%** | 70% |
| **+125% ～ +149.9%** | 60% |
| **+150% ～ +174.9%** | 50% |
| **+175% ～ +199.9%** | 40% |
| **+200% ～ +224.9%** | 30% |
| **+225% ～ +250.9%** | 20% |
| **+250% 以上** | 10% |

※ UP =max(QQQのATH更新後上昇率,SOXのATH更新後上昇率)
※ 「ATH更新後もまず50%上昇までは絶対に削らない」点に注意

## 3. 運用ロジックの要点（クオンツ・アナリストによる補足）

1. **なぜ「目標」ではなく「上限（Cap）」なのか:**
   - 不必要な買い増しによる「高値掴み」を避けるためです。
   - 大底で仕込んだSOXLの爆発力を、上昇局面でも「上限に達するまで」決して手放さないことで、CAGRを最大化します。
2. **なぜ「放置」が戦略に含まれるのか:**
   - どんな下落率から回復しても、市場が過熱（上限値の低下）するまでは、現在持っている強気ポジションを維持するのが数学的に最もリターンを高めるからです。
3. **実運用の極意:**
   - 迷ったときは**「今のSOXL比率は設定された上限を超えているか？」**だけを自問してください。Noなら、市場がどんなに上がっていようが無視して放置。Yesなら、その上限まで機械的に利確する。これだけで感情の介入を完全に排除できます。

## 運用例
資産:1000万円

* 平常時
```
QLD 90%
SOXL 10%

QLD 900万
SOXL 100万
```

* 第1段階①
想定:DD=-12% (QLD -20%, SOXL -40%)
```
総資産は780万円
QLD 900万 → 720万
SOXL 100万 → 60万
```

ルール適応: SOXL比率20%

```
総資産780万円
QLD 624万円
SOXL 156万円
※SOXLを96万円買い増し、QLDを96万円売却
```

* 第1段階②
想定:DD=-22%

```
総資産　780万→650万
QLD: 499
SOXL: 94
```
